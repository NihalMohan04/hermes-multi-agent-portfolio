# Hermes Multi-Agent Portfolio

A modern, responsive portfolio website built entirely by autonomous AI agents using Hermes Agent with model-tiered cost optimization.

## Build Process
- **Orchestrator**: Kimi K2.6 (architect & loop controller)
- **Implementers**: Tiered model assignment (DeepSeek V4 Flash, Qwen 3.5 Plus, Kimi K2.5, etc.)
- **Reviewers**: Automated spec compliance + code quality review via subagents
- **Verification**: Browser screenshots, console error checks, responsive testing

## Design Philosophy
- Clean animations (no bloat, 60fps smooth)
- Fast UI (minimal reflows, CSS-first animations)
- Aesthetic dark theme with glassmorphism and subtle gradients
